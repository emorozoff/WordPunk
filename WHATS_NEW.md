# WordPunk Neo v2.0 — Что нового

## Обзор
Полный редизайн WordPunk в стиле iOS/Telegram: чистый минимализм, плавные анимации, тёмная и светлая тема. Название: **WordPunk Neo**.

## Что сделано

### 1. Дизайн-система
- **Полный CSS-rewrite**: 1800 строк монолитного файла разбиты на 8 модульных файлов
- **Токены**: CSS custom properties для всех цветов, размеров, теней, радиусов, шрифтов
- **Шрифты**: системный стек SF Pro вместо IBM Plex Mono (быстрее грузится, нативнее выглядит)
- **Акцент**: неоновый зелёный сохранён (#00ff88 в тёмной, #00cc6a в светлой)

### 2. Тёмная / Светлая тема
- **ThemeProvider** с React Context
- **3 режима**: Авто (по системе), Светлая, Тёмная
- Переключатель в Настройках — тап для циклического переключения
- Автоматическая синхронизация с `prefers-color-scheme`
- `meta[theme-color]` обновляется динамически

### 3. Дерево навыков (Skill Tree)
- Каждая тема (Кухня, Аэропорт, Работа...) — это узел в дереве
- **Прогресс-бар** и счётчик слов для каждого узла
- **Уровневые пипки** (milestones: 10, 25, 50, 100, 200, 500 слов)
- Завершённые темы выделяются подсветкой
- Общий счётчик знаний наверху (X / Y слов изучено)
- Секции: Основа, Темы, 18+
- Доступно через bottom nav и тап по прогресс-бару

### 4. Нейтральный тон UI
- Все заголовки: Title Case вместо CAPS_PUNK
- Убран весь сленг, мем-язык, дерзкий копирайт
- Вежливый нейтральный тон как в Telegram
- Убраны CRT-сканлайны, глитч-анимации

### 5. Анимации
- Spring-кривые (`cubic-bezier(0.32, 0.72, 0, 1)`) на всех переходах
- Pop-in для попапов, fade-in-up для карточек
- Blur-backdrop на модальных окнах (iOS-стиль)
- Плавные transitions на кнопках, картах, элементах

### 6. Навигация
- **3 таба** в bottom nav: Навыки | Настройки | Статистика
- Тап по прогресс-бару → Skill Tree
- Bottom sheet модалки с spring-анимацией

## Архитектура CSS

```
src/styles/
  global.css      — импорт всех модулей
  tokens.css      — CSS variables для обеих тем
  base.css        — reset, body, анимации
  layout.css      — app shell, header, nav, level bar
  card.css        — flashcard, word card, examples
  options.css     — answer buttons, manual input, mini-history
  modals.css      — bottom sheets, popups, level-up
  screens.css     — stats, settings, topics, debug
  skilltree.css   — skill tree component
```

## Что сохранено
- SRS-алгоритм (5 уровней, интервалы)
- Вся база слов (~7900 карточек, 22 темы)
- R2 CDN аудио (озвучка слов и предложений)
- Режимы озвучки (слово / фраза / выкл)
- Long-press архивация
- Ручной ввод на финале (lvl 4)
- Система флагов

## Известные ограничения
- Прогресс пользователей не мигрируется (сброс при первом запуске v2)
- Иконки тем: используются lucide-react, потенциально можно заменить на SF Symbols при Capacitor-обёртке
- Skill tree пока read-only (визуализация прогресса), без разблокируемых перков
