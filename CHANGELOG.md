# Changelog

Все заметные изменения в WordPunk документируются в этом файле.

## v0.84 — 2026-04-16
- Ручной ввод слова на финале (lvl 4): текстовое поле + боксы по длине слова
- Подсказка первой буквы через 5 секунд
- Автопроверка при заполнении всех позиций + Enter / кнопка ПРОВЕРИТЬ
- Толерантность к опечаткам: Levenshtein <= 1 (для слов длиннее 3 букв)
- Правильно -> архив, неправильно -> уровень -1 с показом верного написания
- В sentence-first карточках целевое слово заменяется на _____
- Дебаг-кнопка «текущую -> lvl 4» для тестирования ручного ввода
- Fix: scrollIntoView при фокусе input (iOS keyboard перекрывал боксы)
- Fix: Unicode smart quotes -> ASCII в JSX (сломанный билд)

## v0.83 — 2026-04-16
- SRS v2: 5 уровней (0-4), интервалы [0, 1, 3, 7, 14] дней
- Уровень 0 -> 1 при 2 правильных подряд (было 3)
- Модификатор сложности: difficulty 1-2 дает +2 уровня за правильный ответ, иначе +1
- Ошибка на уровнях 1-4: -1 уровень (минимум 1)
- Дробный счетчик «знаю слов»: level * 0.2 (1->0.2, 2->0.4, 3->0.6, 4->0.8), архив = 1.0
- StatsScreen: единый getKnownCount() вместо отдельной формулы
- Распределение по уровням: 0-4 (убран lvl 5)
- Убран оптимистичный +1 в счетчик (источник истины только DB)
